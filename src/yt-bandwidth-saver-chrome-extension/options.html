Youtube bandwidth saver

How it works

For everything to work be sure to save the settings and start the local docker container. If you don't know how to start a docker container here's a guide to get started.
  • Visit any youtube video and the extension activates
  • If the video hasn't been downloaded the red extension icon stopped appears, click it to start downloading the video
  • The extensions communicates with the local docker container, that spawns a youtubedl container that downloads the video to your local folder, while downloading the yellow icon appears
  • When the download is complete the extension icon changes to green , and the browser plays your local copy (if you have enabled local playback)
  • The next time you play the same youtube video the extension automatically plays the local copy, you can see that by the icon
